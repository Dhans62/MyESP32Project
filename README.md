# ESP32 BLE OTA Project

Proyek ini menggunakan GitHub Actions untuk kompilasi otomatis firmware ESP32-C3 dan ESP32-S3.

## Spesifikasi
- Core: Arduino-ESP32 v3.0.7
- Fitur: BLE OTA (Over The Air) Update
- Output: File .bin tersedia di tab Actions

## Cara Penggunaan
1. Edit file `.ino` sesuai kebutuhan.
2. Push ke GitHub.
3. Buka tab **Actions** > **ESP32 Core 3.0.7 Build**.
4. Download file dari bagian **Artifacts**.
